# Charity Driver 🧡

**AI-powered platform connecting donors to unhoused individuals across Oakland, CA.**

Built by Paris Irving — [The Hidden Genius Project](https://hiddengeniusproject.org)

## Live Demo
> Hosted on GitHub Pages — link coming soon

## What It Does
- Interactive map of Oakland homeless hotspots with AI-powered insights
- Free-roam donor signup and dashboard
- Delivery logging and impact tracking
- AI-recommended delivery zones based on time of day

## Tech Stack
- Frontend: HTML, CSS, Vanilla JavaScript
- Map: Leaflet.js + OpenStreetMap
- Auth/Database: Supabase (coming in v2)
- Hosting: GitHub Pages

## Pages
- `index.html` — Landing page
- `map.html` — Oakland hotspot map
- `guest.html` — Guest map view
- `signup.html` — Account creation
- `dashboard.html` — Donor dashboard
- `deliver.html` — Log a delivery
